APGI Psychological States Visualization

About This Tool

This web interface provides interactive visualization of psychological states based on the Active Posterior Global Integration (APGI) framework.

To use: Run the Python script with the --demo flag to generate visualizations, then select them from the preview panel.

Visualization Controls

Preview & Information

Available Visualizations

  • Single State Radar: Display parameters for one psychological state
  • State Comparison: Compare multiple states side-by-side
  • 3D State Space: Interactive 3D projection of all states
  • State Transition: Animate transitions between states
  • Category Heatmap: Heatmap visualization for state categories
  • Network Visualization: Interactive network graph of state relationships
  • State Evolution: Animated evolution between states
  • Parameter Sensitivity: Sensitivity analysis for state parameters
  • t-SNE Space: Dimensional reduction visualization

Instructions

  1. Select visualization type from the dropdown
  2. Choose states (use Ctrl/Cmd for multiple selection)
  3. Set output directory
  4. Click "Generate Visualization"
  5. Run: python APGI-Psychological-States-qwen.py --demo to create files
Status: Ready - Run the Python script with --demo to generate visualizations

Python Command Reference

Generate Demo Visualizations:

python APGI-Psychological-States-qwen.py --demo

Launch This GUI:

python APGI-Psychological-States-qwen.py --gui

Get Help:

python APGI-Psychological-States-qwen.py --help
