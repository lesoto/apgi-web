The APGI Framework

A sophisticated scientific computing platform for understanding consciousness through computational modeling and experimental validation

The APGI (Active Passive Inference) Framework implements the Integrated Predictive Ignition framework for testing theories of consciousness through rigorous falsification testing. Built on Active Inference and Predictive Coding theories, it provides comprehensive tools for consciousness research with multi-modal data integration and real-time analysis capabilities.

Core Functionality

Consciousness Modeling & Testing

  • APGI Agent Implementation: Core computational models with interoceptive and exteroceptive processing
  • Dynamic Threshold Mechanism: Models conscious access through precision-weighted prediction errors
  • Somatic Marker Integration: Decision-making based on somatic markers and expected free energy
  • Phase Transition Models: Conscious vs unconscious processing modes
