# APGI Framework Website - Complete Site Guide

The APGI Framework website is a comprehensive consciousness assessment and psychological research platform that helps users understand their cognitive architecture through scientifically-grounded measurements. This guide describes every page and its functionality from a user perspective.

## 🧠 Core Concept

The APGI Framework moves beyond traditional personality testing by measuring four key parameters of consciousness:

- **ε (Epsilon)**: Environmental Sensitivity - How deeply you feel the world
- **π (Pi)**: Processing Intensity - How complex your thought patterns are
- **θₜ (Theta-t)**: Awareness Threshold - How much stimulus before you notice
- **β (Beta)**: Body-World Balance - Where you focus internally or externally
