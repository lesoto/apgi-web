The Central Problem

The Bistable Consciousness Paradox

Identical physical inputs — same photons, same decibels — produce radically different conscious outcomes across trials. Existing models have no principled account. APGI resolves this through liquid network attractor dynamics.

The Paradox

Same Stimulus, Different Outcome

In bistable rivalry and near-threshold masking paradigms, physically identical stimuli produce conscious percepts ~50% of trials. No prior model explains trial-to-trial variability without invoking pure noise.

The Resolution

Phase Transition in High-D State Space

Liquid networks operate near a critical point. Small fluctuations in precision weights or threshold state tip the system into different attractor basins — deterministic chaos, not random noise.

Previous Accounts

Static Threshold Models Fail

Fixed-threshold accumulator models cannot explain why the same accumulated evidence sometimes ignites consciousness and sometimes does not. They lack a dynamic reservoir with state-dependent history.

APGI Mechanism

Fading Memory + Nonlinear Separability

The cortical liquid's fading memory of prior states shapes which inputs push the system over threshold. Precision-gated gain modulates input strength; the reservoir's current state determines ignition probability.

Formal Specification

The Complete ODE System

dxi/dt = −xi + Σj wij·σ(xj) + Σk wikin·uk
The core reservoir ODE. xi is neuron activation, wij are fixed random recurrent weights, σ is a nonlinear activation, and uk are precision-modulated inputs.
wikin(t) = w₀ · Πk(t)
Input weights are dynamically scaled by precision Π, implementing the APGI gating mechanism. High precision amplifies input gain; low precision suppresses it below ignition threshold.
Esynapse ~ Π · θ
Metabolic cost scales with both precision weighting and threshold magnitude. High-precision, high-threshold conscious access is metabolically expensive — exactly as empirically measured (5–10% glucose premium).
