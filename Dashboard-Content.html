1. TikTok Master Template

  • Hook (0-3 seconds):
    • → Visual Punch: Color flash/zoom effect for immediate attention
    • → Text Overlay: Bold, 72pt Sans-serif text with white fill and black outline
    • → Audio: Trending sound from TikTok library (sources documented)
  • Main Content (3-45 seconds):
    • → Scene 1: Concept introduction (5-10 seconds)
    • → Scene 2: Examples and proof points (10-15 seconds)
    • → Scene 3: Practical takeaway/application (10-15 seconds)
    • → Scene 4: Quick summary and transition (5-10 seconds)
  • CTA (45-60 seconds):
    • → Text: "Link in bio" call-to-action overlay
    • → Transition: Animated end screen with brand elements
    • → Audio: Ending audio tag or sound effect
    Technical Specifications:
    Resolution: 1080x1920 (9:16 vertical)
    Frame Rate: 30fps
    Audio Levels: -3dB for optimal platform performance
    Text Style: Sans-serif, white text with black outline for readability
    Music: Trending audio from TikTok library (all sources documented)
    Duration: 15-60 seconds (optimal: 21-34 seconds)
    File Format: .mp4 (H.264 codec)
    Color Space: sRGB for mobile display consistency
    Hook Variations:
    Flash/Zoom/Cut
    Content Types:
    Educational/Entertainment
    CTA Styles:
    Direct/Soft/Interactive
    Audio Options:
    Trending/Custom/Voiceover
    APGI_YouTube_Longform_Master.prproj (Premiere Pro)

    2. YouTube Long-Form Master

  • Intro Sequence (0-90 seconds):
    • → Channel Intro: 3-second animated branding sequence
    • → Hook Statement: 10-second compelling opening question/claim
    • → Value Proposition: 15-second what viewer will learn/achieve
    • → Video Outline: 30-second roadmap of content structure
    • → Subscribe CTA: 22-second call to subscribe with benefits
  • Chapter 1: Problem (90-300 seconds):
    • → Visual B-roll: Contextual footage and demonstrations
    • → On-screen Graphics: Key points and data visualization
    • → Voiceover: Professional narration with proper pacing
    • → Music Bed: Background audio at appropriate levels
  • Chapters 2-4: Solution & Application:
    • → Structure: Repeat chapter format for consistency
    • → Content: Progressive complexity and practical application
    • → Engagement: Interactive elements and viewer prompts
  • Conclusion (last 180 seconds):
    • → Key Takeaways: 30-second summary of main points
    • → Call to Action: 30-second specific next steps for viewer
    • → Related Videos: 30-second playlist recommendations
    • → Subscribe Reminder: 60-second final engagement push
  • End Screen (last 20 seconds):
    • → Next Video: Recommended content with thumbnail
    • → Channel Subscription: Prominent subscribe button
    • → Playlist Suggestion: Curated content series
    Technical Specifications:
    Resolution: 1920x1080 (16:9 landscape)
    Frame Rate: 30fps
    Audio Levels: -6dB dialogue, -12dB music, -18dB SFX
    Color: APGI brand colors (reference in project)
    Fonts: Inter (sans-serif), Crimson Pro (serif accents)
    Duration: 8-20 minutes optimal
    File Format: .mp4 (H.264 codec)
    Chapters: Timestamped for navigation

    4. Facebook Ad Master

  • Single Image Ad (1080x1080px):
    • → Visual: APGI Brain visual centered and prominent
    • → Headline: Maximum 5 words for optimal engagement
    • → Description: Maximum 90 characters for readability
    • → CTA Button: Blue color, 80px height for visibility
  • Carousel Ad (5 cards, 1080x1080px each):
    • → Card 1: Hook image with compelling visual
    • → Card 2-4: Benefit slides with clear value propositions
    • → Card 5: CTA + offer with clear call-to-action
    • → Captions: Platform-specific text for each card
  • Video Ad (1080x1920px vertical OR 1280x720px horizontal):
    • → 3-second Attention Grab: Immediate visual or audio hook
    • → 15-second Value Demonstration: Core benefit showcase
    • → Final 2 seconds: CTA + logo for brand recall
    Technical Specifications:
    File Format: .psd with organized layer structure
    Color Profile: sRGB (web standard)
    Text: Arial or system font (Facebook default)
    Brand Elements: Logo + brand colors throughout
    Resolution: Multiple formats for different placements
    Compression: Optimized for web loading speed
    Guidelines: Facebook ad policies compliant
    Single Image:
    1080px Square, 5-word headline max
    Carousel:
    5 Cards (Hook → Benefits → Offer)
    Video:
    3s Grab → 15s Demo → 2s CTA/Logo
    Formats:
    Vertical/Horizontal options

    5. Quote Graphics

  • 1. Square Quote (1080x1080px - Instagram Feed):
    • → Background: Dark with APGI Brain watermark
    • → Quote: 30-50 characters, white text for contrast
    • → Attribution: Author + source clearly credited
  • 2. Story Quote (1080x1920px - Instagram Story):
    • → Background: Full-bleed color gradient
    • → Quote: 40-60 characters, centered placement
    • → Swipe-up CTA: "Tap for more" interactive element
  • 3. Pinterest Pin (1000x1500px - Pinterest):
    • → Orientation: Vertical format for Pinterest feed
    • → Quote: 40-60 characters + context (25 characters)
    • → Colors: 3-4 brand colors maximum for consistency
  • 4. Facebook Post (1200x628px - Facebook Feed):
    • → Format: Horizontal layout for Facebook feed
    • → Quote: 40-60 characters for optimal readability
    • → Border: 10px APGI electric blue for brand recognition
  • 5. LinkedIn Article (1200x627px - LinkedIn):
    • → Aesthetic: Professional design for business audience
    • → Content: Quote + source attribution for credibility
    • → Branding: LinkedIn icon in bottom corner
    Multi-Platform Readiness
    Design Guidelines:
    Platform Optimization: Each format sized for specific social platform
    Brand Consistency: APGI colors and elements throughout
    Typography: Readable fonts with proper hierarchy
    Character Limits: Optimized for platform best practices
    File Formats: PNG for quality, JPG for compression
    Accessibility: High contrast for screen readers

    6. Chapter Summary Infographic

  • Format: 1080x1920px (Instagram Story friendly)
  • Header (10% height):
    • → Chapter Number: Clear numerical identification
    • → Chapter Title: Descriptive heading for context
    • → APGI Branding: Logo and color scheme
  • Main Content (60% height):
    • → Key Concept Visualization: Central graphic representation
    • → 3-4 Bullet Points: Maximum 20 characters each for readability
    • → Visual Icons/Diagrams: Supporting graphics for each point
    • → Color-Coded Parameters: Visual hierarchy with APGI colors
  • Footer (30% height):
    • → QR Code: Links to full chapter content
    • → "Tap for more" Text: Interactive call-to-action
    • → APGI Branding: Logo and website URL
    Technical Specifications:
    Resolution: 1080x1920px (Instagram Story friendly)
    Format: Vertical orientation for mobile viewing
    Color Scheme: High contrast for accessibility
    Typography: Clear hierarchy with readable fonts
    Interactive Elements: QR code and tap-to-action
    Brand Consistency: APGI colors and logo placement
    File Format: PNG for quality, optimized for web
    Layout:
    Header 10% | Main 60% | Footer 30%
    Content:
    Visual + Text elements
    Interaction:
    QR code + CTA
    Platform:
    Instagram Story optimized
