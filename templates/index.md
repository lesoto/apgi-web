## Brand Guidelines

### Colors
- **Primary Blue:** #1e40af
- **Secondary Teal:** #0d9488
- **Accent Purple:** #7c3aed
- **Warning Orange:** #f59e0b
- **Text Dark:** #0f172a

### Typography
- **Headlines:** Space Grotesk (700 weight)
- **Body Text:** Inter (400-500 weights)
- **Accent Text:** Playfair Display (for quotes)

## Platform-Specific Best Practices

### Facebook
- Post dimensions: 1200x630px (standard)
- Character limit: 63,206 characters
- Hashtag usage: 3-5 relevant hashtags
- Best posting times: 2-4 PM weekdays

### Instagram
- Post dimensions: 1080x1080px (square)
- Character limit: 2,200 characters
- Hashtag usage: 5-10 relevant hashtags
- Best posting times: 9-11 AM, 2-4 PM

### YouTube
- Thumbnail dimensions: 1280x720px
- Title limit: 100 characters
- Description: Detailed with keywords
- Best posting times: 2-4 PM, 7-9 PM

### TikTok
- Video dimensions: 1080x1920px
- Video length: 15-60 seconds optimal
- Hashtag usage: 3-5 hashtags
- Best posting times: 7-9 PM

### Telegram
- Image limit: 10MB per file
- Video limit: 50MB per file
- Character limit: 4,096 characters
- Posting frequency: 1-3 times daily
