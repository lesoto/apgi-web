## Usage Instructions

### For Video Creators
1. Select appropriate template for content type
2. Customize text and imagery while maintaining brand consistency
3. Test thumbnails on mobile devices
4. A/B test different thumbnail variations
5. Follow YouTube's community guidelines

### For Graphic Designers
1. Maintain safe zones for all template types
2. Use high-resolution images (minimum 1280px width)
3. Export in PNG format for quality
4. Keep file sizes under 2MB for thumbnails
5. Test designs at different sizes

### Technical Specifications

#### Thumbnail Requirements
- **Format**: JPG, PNG, BMP, or GIF
- **Resolution**: 1280x720px (recommended)
- **Aspect Ratio**: 16:9
- **File Size**: Under 2MB
- **Safe Area**: 85% of frame to avoid cropping

#### Banner Requirements
- **Format**: JPG or PNG
- **Resolution**: 2560x1440px
- **File Size**: Under 6MB
- **Safe Zones**: Mobile and desktop viewing areas

#### End Screen Requirements
- **Duration**: 5-20 seconds
- **Elements**: Up to 4 elements
- **Placement**: Final 20% of video
- **Resolution**: Match video quality
