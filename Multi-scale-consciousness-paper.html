Multi-Scale Consciousness:
Hierarchical Temporal Depth

Extending APGI to Five Nested Levels — from Millisecond Feature Binding to Autobiographical Identity

Conscious experience does not happen at a single timescale. It unfolds simultaneously across sub-second feature binding, second-to-minute deliberation, and years-long autobiographical identity. This paper extends APGI to five nested hierarchical levels, proposing that cortical hierarchies implement the same precision-gating logic at progressively longer timescales — explaining how disparate temporal dynamics cohere into unified experience.

Hierarchical Architecture

Five Levels of Conscious Time

The same Π·|ε| > θ logic operates at each level — but with level-specific timescales, neural substrates, and threshold calibrations.

L1

Feature Binding

Rapid integration of sensory features — color, motion, form — into unified percepts. Implemented via gamma oscillations and local cortical synchrony. Fails in visual agnosia.

τ ≈ 10–100 ms
L2

Perceptual Integration

Cross-modal binding and short-term sensory buffering. The alpha/beta oscillation band bridges features into coherent perceptual objects across modalities.

τ ≈ 100–500 ms
L3

Deliberation Window

The specious present — the unified 1–3 second window of "now" that William James described. Theta oscillations gate working memory, enabling active deliberation.

τ ≈ 0.5–5 s
L4

Goal & Planning

Multi-step planning and goal maintenance across minutes. Prefrontal slow oscillations and hippocampal theta sequences implement prospective temporal coding.

τ ≈ seconds–minutes
L5

Autobiographical Identity

The self across time — narrative identity, semantic memory, long-horizon planning. Default mode network slow fluctuations sustain the temporal scaffolding of selfhood across years.

τ ≈ hours–years
Temporal Phenomenology

The Specious Present

1–3 seconds
The Window of "Now"
William James (1890) called it the specious present — the brief interval that feels like a unified moment. APGI localizes this to Level 3 integration: the theta-band window where prediction errors, working memory content, and deliberative processes converge into a single conscious moment before passing into the past.
Cross-Frequency Coupling

How Levels Talk to Each Other

The five levels are not independent. Phase-amplitude coupling allows slow oscillations to modulate the amplitude of faster ones — enabling top-down temporal context to gate bottom-up sensory precision.

Clinical Implications

Disorders as Temporal Dyscoherence

Psychiatric conditions can be reframed as failures of temporal coherence across levels — with specific APGI parameters as intervention targets.

PTSD

Temporal Dyscoherence

Past traumatic events intrude into Level 3 (specious present) with excess precision, preventing normal L3–L4 boundary maintenance. The past feels like the present.

Depression

Temporal Flattening

L4–L5 temporal depth collapses — the future becomes inaccessible. Depressed subjects show reduced delta-theta coupling, flattening the prospective planning window.

Dissociation

Level Fragmentation

L1–L2 perceptual binding fails to propagate to L3 deliberation. Percepts are processed but not integrated into the unified specious present — derealization, depersonalization.

Autism

Temporal Atypicality

Atypical CFC ratios — particularly reduced theta-gamma coupling — alter the temporal resolution of the specious present, producing different but not deficient temporal binding.
